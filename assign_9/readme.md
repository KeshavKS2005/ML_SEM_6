# Assignment 9 - Human Activity Recognition (HAR) using UCI Dataset

## Overview
This assignment works with the UCI Human Activity Recognition (HAR) dataset. It includes data preprocessing, feature extraction, and classification of activities using various machine learning models.

## Included Files
- `assignment9..ipynb` – Jupyter notebook containing the full analysis pipeline.
- `ex9.ipynb` – Alternative notebook with additional experiments.
- `UCI HAR Dataset.zip` – Compressed raw dataset (to be extracted).
- `UCI HAR Dataset/` – Extracted dataset folder containing training and test data.
- `UCI HAR Dataset.names` – Description of the dataset features.

## How to Run
1. **Extract the dataset**: Unzip `UCI HAR Dataset.zip` to obtain the `UCI HAR Dataset` folder.
2. **Install dependencies**: `pip install -r requirements.txt` (if a requirements file is present) or ensure you have `numpy`, `pandas`, `scikit-learn`, `matplotlib`, and `seaborn` installed.
3. **Open the notebook**: Launch Jupyter Notebook (`jupyter notebook`) and open `assignment9..ipynb` (or `ex9.ipynb`).
4. **Run all cells**: Execute the notebook sequentially to load the data, perform preprocessing, train models, and evaluate performance.
5. **Review results**: The notebook will generate classification reports, confusion matrices, and visualizations of the activity predictions.
